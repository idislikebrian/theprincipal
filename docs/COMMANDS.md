# Command list
><> = required, [] = optional

## Active
|	Command	| description	| Usage
|---------------|--------------------|--------------|
| poll	|	Create a poll for users to answer.	|	`poll <question>`   |
| purge	|	Deletes a n amount of messages.	|   `purge <n>` |
| say	|	Sends an announcement.	|	`say <announcement>`    |
| music |   Plays music in your current voice channel | `music <YouTube link>` OR `music <search term>` |
